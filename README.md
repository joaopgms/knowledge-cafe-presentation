## Running the HTML deck

Just double-click `kc_presentation_may2026.html` — it opens in your default browser. No build step.

### Keyboard

| Key | Action |
|---|---|
| `→` / `Space` | Next slide |
| `←` / `Backspace` | Previous slide |
| `F` | Fullscreen (resets to slide 1, plays the 10-second sting) |
| `T` | Toggle dark / light theme |
| `M` | Toggle music manually |
| `Home` / `End` | Jump to first / last slide |

Click the right half of the screen to advance, left half to go back. Swipe on touch.

### Stage-day audio note

The browser plays through whichever audio device the OS has set as default. Before going live, disconnect any Bluetooth earphones and confirm the laptop / venue speakers are the default output.
